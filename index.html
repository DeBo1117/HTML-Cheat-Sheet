HTML Cheat Sheet


Table Content

Tag Name Description
<table> Table Used contain all tabular data.
<thead> Table Heading Used to add titles to rows & columns.
<tbody> Table Body Allows long tables to be sectioned off.
<tr> Table Row Used to create table rows.
<td> Table Data Used to add cells to contain data in table.
<tfoot> Table Footer Used to section off long table at bottom.



Forms

Tag Name Description
<input> input Creates single row for user input.
<textarea> textarea Creates a bigger text field for user input. Can be expanded with rows & columns.
<form> form Resonsible for collecting info to send somewhere else.
<datalist> datalist Creates text field users can type info & filter options.
<label> label For users to properly indentify <input>.



Semantic HTML

Tag Name Description
<header> Header Container for either nav links or intro content containing .
<nav> Nav Used to define block of navigation links (menues, table of contents) inside of <header>.
<main> Main Used to encapulate dominant content within webpage.
<footer> Footer Used to contain info such as Contact Info, Copyright, Terms of Use, Site Map, Reference to top of page links.
<article> Article Used to hold content such as articles, blogs, magazine etc.
<section> Section Defines elements in docments like chapters, headings.
<aside> Aside Used mark additional info that can enhance another element but isn't required in order to understand main content.
<figure> Figure Used to encapsulate media such as image, illustration, diagram, code snippet etc.
<figcaption> Figcaption Used inside of <figure> element to descibe image. Helps group <figure> content.
<videon> Video Used to add videos to website. Some attributes that can alter playback include: controls, autoplay, loop.
<embed> Embed Used to embed any media content (videos, audio links, gifs).
